Unbeatable Tictactoe

A tictactoe game made with React and Material-UI, it boasts the following features:

  • Fully Responsive Design
  • Option to play on 3 x 3, 5 x 5 or 7 x 7 grids
  • Play against an unbeatable computer or play against a friend locally
  • Variety of icons with customisable colours

FaceBrain

This tool uses Clarifai's face recognition API to find faces in submitted URLS. Features include:

  • Fully Responsive Design
  • A PostgreSQL database to store login details and usage statistics
  • A Node/Express API for retreiving user profiles from the database and accessing Clarifai's API
  • Password hashing and authentication with Bcrypt.js
